Using em

The default value of em is 16px.
It is relative to the parent's font.

Using rem

The default value of rem is also 16px.
It is relative to root of html.
Root of html can be changed by writing below syntax inside style
html{ font-size:10px;//changing the default of Rem i.e 1rem=10px; }
